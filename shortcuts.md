# Adrishya - Keyboard Shortcuts Reference (`shortcuts.md`)

This guide lists all the global keyboard shortcuts and interactive window controls supported by **Adrishya** to make your workflow seamless during online meetings or tests.

---

## 1. Global Keyboard Shortcuts (System-Wide)

These shortcuts function **globally** across your entire operating system. You can trigger them from *any* active window (e.g. while focused on a browser tab with Google Meet, or writing code in VS Code).

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| **`Ctrl` + `Shift` + `A`** | **Toggle Visibility** | Instantly hides the overlay window from view, or shows and focuses it. |
| **`Ctrl` + `Shift` + `V`** | **Toggle Voice Recording** | Instantly starts or stops recording your voice for speech-to-text. |
| **`Ctrl` + `Shift` + `Q`** | **Quit Application** | Instantly terminates the application processes and closes all windows. |

---

## 2. Window & Application Controls (Mouse-Based)

| Element | Interaction | Action |
| :--- | :--- | :--- |
| **Header Area (Logo/Title)** | Click & Drag | Smoothly relocates the frameless overlay window anywhere on your desktop. |
| **Minimize / Collapse Icon** | Click | Collapses the full panel into a compact, horizontal pill widget. |
| **Maximize / Expand Icon** | Click | Expands the compact pill back into the full interactive panel. |
| **Trash / Clear Chat Icon** | Click | Instantly clears your current conversation history. |
